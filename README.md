# Spring Relaciones (Java 21)
## Instrucciones
1. Importa el proyecto en Spring Tools Suite (File -> Import -> Existing Maven Project).
2. Asegúrate de tener MySQL ejecutándose y crea la base `spring_relaciones`.
3. Ejecuta la clase `SpringRelacionesApplication` como Spring Boot App.
4. Abre `http://localhost:8080/` en el navegador.
